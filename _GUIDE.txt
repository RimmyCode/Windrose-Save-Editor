# =====================================
# Prerequisities:
# =====================================
# For every item you want to replace, have something to overwrite it with in your inventory.
#
# For example if i want to give myself the Mighty Boar Whistle, ill put a dodo egg or any worthles item in my inventory.
#
# Close your game while save editing.
#
# =====================================
# Usage:
# =====================================
#   python windrose_save_editor.py "<path to Players\<GUID> folder>"
#
# Example:
#   python windrose_save_editor.py "C:\Users\YOU\AppData\Local\R5\Saved\SaveProfiles\<SteamID>\RocksDB\0.10.0\Players\<GUID>"
#
# The script will auto-detect the correct save folder if you point it
# at a parent directory (e.g. just the SteamID folder).
# 
# To give yourself an item, first find its ItemParam by using the Item ID Guide (item_guide.html). 
# Click on the item you want or the "Add to save" Button to open a window, then copy the very first text box "ItemParams" you will need this in a moment.
#
# When youre successfully in the save editor inside your terminal, press option 4 (Replace item)
# Choose the item you wish to overwrite (IT WILL BE DELETED), then paste the "ItemParams" text you copied into the item param text box.
#
# Continue as instructed, for module i recommend 0 as that is your backpack (NOT HOTBAR)
#
# Once you are happy with your changes, press use option 7 to save.
#
# Load up your game and confirm everything is working.
#
# =====================================
# Item ID Guide (item_guide.html):
# =====================================
#   Standalone HTML - no install needed, opens in any browser.
#   To regenerate from your own FModel export:
#
#   python "parse_items.py" "<FModel export path>" "./output"
#
#   Place your locres file as "game.json" anywhere inside the export folder
#   for proper item names (export Game.locres from FModel as JSON).
